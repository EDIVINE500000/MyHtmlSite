## MyHtmlSite
🍔 Vinny Restaurant - Smart Food Ordering System

A simple and modern online food ordering system developed using HTML5, CSS3 (Inline Styling), and Vanilla JavaScript. The system allows customers to register, log in, select food and drinks, apply promo codes, and make payments through mobile money services available in Tanzania.

---

✨ Key Features

* Login and Registration Page (Auth Tab Switcher): Customers can securely create an account and log in using JavaScript form validation.
* Vinny Restaurant Section: The restaurant name is professionally displayed above the registration form.
* Interactive Dashboard Summary Widgets:
  * Current Transaction: Displays the real-time total cost of selected items.
  * Discount: Shows the amount deducted after applying a promo code.
  * Delivery Status: Tracks whether a delivery location has been selected and whether the order is being prepared or delivered.
* Dynamic Menu Selection: Includes a "+ Show More Menu" feature to hide or display additional menu items for a cleaner interface.
* Promo Code System: Provides an automatic 20% discount using the special code ("MZUMBE20") when the order exceeds TSh 10,000.
* Mobile Money Payments: Supports Vodacom M-Pesa, Tigo Pesa, and Airtel Money.
* Feedback Section: Customers can leave feedback after receiving the service.
